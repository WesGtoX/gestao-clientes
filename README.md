### Install and Run

1. Clone the repository:
```bash
git clone https://github.com/wesgtox/gestao_clientes.git
```
2. Install the dependencies:
```bash
poetry install
```
3. Run:
```bash
poetry shell
python manage.py migrate
python manage.py runserver
```
